клиент не может создавать объекты, только выражать желание

приветственное сообщение (сервер должен создать танк до отправления инфы):
Vector2i - размер уровня
int колво объектов в мире
конструкторы всего + сразу инфа о них
int - номер объекта, являющегося танком игрока

объекты:
Bonus(BonusType bt, int value, sf::Vector2f pos)
Tank(sf::Vector2f pos)
MissileSniper(sf::Vector2f pos, float angle, b2Vec2 tankLinVel)
MissileSimpleBomb(sf::Vector2f pos, float angle, b2Vec2 tankLinVel)
Planet(sf::Vector2f pos, int radius, int power)

пакет обновления мира:

4байта - количество блоков, содержащих инфу о новых объектах
[4 байта - тип объекта
4 байта - номер объекта
далее - соотв инфа для конструктора объекта]

4 байта - количество блоков, содержащих инфу о существующих объектах
[4 байта - номер обновляемого объекта
? байт - позиция x,y
? байт - линейная скорость x, y
? байт - угол
? байт - угловая скорость]

Пакет от клиента:
float - угол дула
byte - хочу выстрелить
